## 🗺️ ¿Qué ofrece esta edición?

Atlas-TW Sequito Edition es la guía definitiva de exploración y botín:

- **Mapas Turtle WoW**: Inclusión de todas las nuevas zonas y mazmorras (ej. Gilneas, Hyal, Emerald Dream).
- **Loot de Raid**: Base de datos de objetos actualizada con las tablas de botín de las bandas del clan.
- **Integración pfQuest**: Sincronización de pines de misiones directamente sobre los mapas de Atlas.
- **Skins de El Séquito**: Interfaz de navegación con texturas oscuras y bordes púrpuras.
- **Localización Completa**: Traducción de todos los nombres de NPCs y Bosses al español.

## 🚀 Instalación & Uso

1.  Extrae en `Interface\AddOns\Atlas-TW\`.
2.  Haz clic en el icono del minimapa o usa `/atlas` para abrir el explorador.
3.  Usa la pestaña de **"Loot"** para planificar tu equipo de fase final.
