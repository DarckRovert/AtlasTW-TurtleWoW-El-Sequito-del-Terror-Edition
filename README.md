## 🏛️ ¿Qué es Atlas-TW?

Es la versión definitiva de Atlas para **Turtle WoW**, que integra mapas de mazmorras personalizadas, tablas de botín actualizadas y misiones de instancia de forma nativa.

### 🧩 Navigator Feature Matrix (Capacidades)

| Módulo | Tipo de Datos | Funcionalidad | Descripción |
| :--- | :--- | :--- | :--- |
| **Loot Tracker** | Database | Botín de Turtle WoW | Tablas de drop actualizadas para mazmorras custom |
| **Quest Linking** | Integration | Misiones de Mazmorra | Muestra qué misiones pertenecen a cada instancia |
| **Instance Map** | Visualizer | Navegación de Bosses | Mapas detallados con ubicación de encuentros |
| **pfUI Sync** | Visual Mod | Skin Apex-Tier | Interfaz armonizada con el ecosistema del Séquito |
| **Search Engine** | Utility | Filtro de Botín | Encuentra equipamiento por nombre en todo Atlas |

## 🚀 Despliegue Rápido (Map-Flow)

1.  **Limpieza**: Borra cualquier versión previa de `Atlas` o `AtlasLoot` en `AddOns/`.
2.  **Instalación**: Extrae en `Interface\AddOns\Atlas-TW\`.
3.  **Activación**: Abre el mapa principal (M) o haz clic en el icono del minimapa de Atlas.
